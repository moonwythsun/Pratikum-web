Pengenalan Web Development

Pertemuan Awal

Pada pertemuan awal mata kuliah pemrograman web, mahasiswa akan mempelajari dasar-dasar pembuatan halaman web statis. Materi difokuskan pada pemahaman struktur dokumen HTML serta peran CSS dan JavaScript dalam pengembangan web.

  • Struktur HTML
  • Styling CSS
  • Interaksi JavaScript
  • Referensi MDN

Struktur HTML

HTML digunakan untuk menyusun elemen-elemen utama pada halaman web seperti teks, gambar, tabel, dan formulir.

  1. Tag dasar
  2. Heading dan paragraf
  3. List dan link

Styling CSS

CSS berfungsi untuk mengatur tampilan visual halaman web agar lebih rapi, menarik, dan responsif terhadap berbagai ukuran layar.

Interaksi JavaScript

JavaScript memungkinkan halaman web menjadi interaktif, seperti menampilkan notifikasi, validasi form, dan manipulasi elemen secara dinamis.
